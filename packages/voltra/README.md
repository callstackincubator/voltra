# Component schema (monorepo)

Internal workspace for Voltra’s shared component schema (`data/components.json`) and the props/code generator.

App developers should install the published client packages documented at [use-voltra.dev](https://use-voltra.dev/getting-started/installation):

- `@use-voltra/ios-client`
- `@use-voltra/android-client`

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for how to run `npm run generate`.
